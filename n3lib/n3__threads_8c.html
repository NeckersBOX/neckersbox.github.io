◆ __n3l_threads_routine()

void* __n3l_threads_routine ( void *  arg)

Internal thread routine to execute the queue routines.

This function run in a separate thread and still alive until n3l_threads_flush() is called. If no element are present in the queue, it waits for a cond signal then start executing one-by-one all elements in the queue.

Note
Each time an element in the queue is picked, the element routine run inside this thread.
